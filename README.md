# LeetCode UI — QA Automation Suite

## 📖 Overview
Automated test suite for **LeetCode** platform UI interactions using Selenium WebDriver, Java, and TestNG. Validates problem search, filtering by difficulty/topic, and navigation workflows on the LeetCode platform.

---

## 🚀 Key Testing Highlights
- **Problem Search:** Validated keyword-based problem search and result accuracy
- **Difficulty Filter:** Tested Easy/Medium/Hard filter combinations
- **Topic Tag Filter:** Validated filtering by data structure and algorithm categories
- **Navigation Testing:** Verified problem page navigation and back-navigation workflows
- **POM Architecture:** Maintainable page object structure for scalability

---

## 🛠️ Tech Stack
| Technology | Purpose |
|---|---|
| Java | Core language |
| Selenium WebDriver | Browser automation |
| TestNG | Test runner & assertions |
| Maven | Build & dependency management |

---

## ▶️ How to Run
```bash
git clone https://github.com/ritvikz/crio-leetcode-qa.git
cd crio-leetcode-qa
mvn test
```
